### Cash Register Assignment
Rails v5, Ruby v2.3.3

Project Goal:
- Create a cash register that can take in and dispense bills of denominations 20, 10, 5, 2 and 1. 
- Given an amount, the register should dispense bills to make that change; if there are insufficient bills, raise an error.

Current Status:
- Assignment complete
- Functional with register
- Tests written for register model
